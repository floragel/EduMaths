# EduMaths
Ce site, nommé Edumaths, est une plateforme éducative dynamique développée en PHP/MySQL. Il permet aux étudiants de consulter et rechercher des cours de mathématiques par thématique dépendamment de leur classe. Le projet inclut une interface d'administration complète pour ajouter, modifier ou supprimer le contenu pédagogique et les enseignants.
